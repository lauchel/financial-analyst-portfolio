# Financial Analyst Portfolio — Acme Co. FY 2024

Work samples demonstrating financial analysis, modeling, and SQL skills.

---

## Files

### AcmeCo_PL_Dashboard_2024.xlsx
A full-year Profit & Loss dashboard for a fictional small business.
Includes monthly revenue, expense breakdown by category, net profit,
gross margin, and a KPI summary. Built with industry-standard color
coding (blue = inputs, black = formulas), named ranges, and dynamic
charts. Demonstrates: financial modeling, Excel formula construction,
controllership thinking.

### AcmeCo_CashFlow_Forecast_2024.xlsx
A 12-month cash flow forecast with dynamic opening/closing balances
that cascade automatically when inputs change. Includes a live buffer
alert that flags any month where cash drops below $15,000, separate
CapEx line items, and a KPI summary. Demonstrates: cash vs. accrual
distinction, forward-looking financial planning, scenario awareness.

### AcmeCo_AR_Aging_Tracker_2024.xlsx
An accounts receivable aging tracker across 20 invoices and 10
customers. Invoices are bucketed into Current / 1-30 / 31-60 / 61-90
/ 90+ day aging categories with color-coded risk levels and
recommended collection actions per customer. Demonstrates: AR
management, controllership, collections prioritization.

### AcmeCo_SQL_Portfolio_Queries.sql
Six analyst-level SQL queries against a financial database covering
revenue trending (LAG window function), expense breakdown (GROUP BY),
profit margin by month (JOIN), AR aging buckets (CASE WHEN), high-risk
customer identification (HAVING), and actual vs. forecast variance
analysis. Demonstrates: SQL for financial reporting, data extraction,
KPI analysis.

---

## About
All data is fictional. Models are built to demonstrate analytical
thinking, financial concepts, and technical execution — not to
represent any real company.
